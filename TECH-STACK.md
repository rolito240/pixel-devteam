# Tech Stack

## Approved Technologies

### Frontend
- React 19+ / Next.js 15+
- TypeScript (mandatory for new projects)
- Tailwind CSS v4
- Vite (for SPAs)

### Backend
- Node.js + Express / Fastify
- Python + FastAPI
- Go (when performance-critical)

### Mobile
- Flutter (primary)
- React Native (secondary)

### Database
- Supabase (primary - PostgreSQL + Auth + Storage + Edge Functions)
- MongoDB (document-heavy use cases)
- SQLite (local/prototyping)

### AI/ML
- DeepSeek API (primary LLM)
- LangChain / LlamaIndex (RAG)
- MCP servers (tool integration)

### DevOps
- Docker + Docker Compose
- GitHub Actions (CI/CD)
- Vercel / Cloudflare (deploy)

### Testing
- Playwright (E2E)
- Jest / Vitest (unit)
- Pytest (Python)

### Design
- Figma (design system)
- Tailwind CSS (implementation)

## Forbidden

- jQuery (legacy)
- PHP (unless client specifically requires)
- MongoDB without schema validation
- Any library without active maintenance (last commit > 2 years)
