# Client Patterns

## Common Request Types
- Landing pages: Most frequent, easiest to estimate
- Admin dashboards: Medium complexity, need clear data models
- E-commerce: High complexity, payment integration critical
- AI chatbots: Growing demand, need clear use case definition

## Pricing Insights
- Colombian market: price sensitive, value speed
- International market: value quality, pay more
- AI services: premium pricing accepted

## Communication Tips
- Always respond within 24h
- Use simple language (avoid jargon)
- Show demos early and often
- Get written approval before starting
